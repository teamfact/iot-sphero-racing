# Introduction

# Pre-requisites 

1. You need to have a Sphero device

2. Xbox 360 controller

# Set things up

Checkout the project:

```
	git clone git@github.com:teamfact/iot-sphero-racing-client.git
```  

Run the followng code to install the required dependencies: 

```
  npm install
```
